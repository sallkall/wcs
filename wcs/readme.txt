What are included:

1) wcs_demo_v7.ipynb: Demo Python notebook for analyzing WCS

2) wcs_demo_v7_table.ipynb: Similar to 1) except allowing some data to be converted to tables (i.e., a data structure developed and used commonly at UCB)

3) wcs_helper_functions.py: Helper functions for analyzing WCS

4) WCS_data_core: a compact/core subset of the WCS data that would be sufficient for most analytical purposes

==================================================================================

Prerequisites for running the demo notebook:

1) Install Python Jupyter Notebook (https://www.continuum.io/downloads): v3.5 recommended

2) Place all files and folders under the same directory
